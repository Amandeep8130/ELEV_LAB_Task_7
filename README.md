 Overview
 
The Image Resizer Tool is a simple yet powerful Python script designed to resize and convert images in batch. Whether you’re preparing images for a portfolio, compressing them for faster loading, or standardizing dimensions for a project, this tool makes the process quick and beginner‑friendly.

Objective
Resizing all the images according to the user input dimenstions of the photo.

Convert images into a the desired size/

Save resized images into a separate output folder for easy organization.

Tools & Technologies

Python 3

Pillow (PIL fork) – Python Imaging Library for image processing
